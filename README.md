# VisualArc Gallery Template

![Preview](https://cdn.poehali.dev/templates/arc-gallery-ru/preview-ru.jpg)

Современный шаблон галереи изображений с уникальным дуговым расположением карточек. Минималистичный дизайн с плавными анимациями и адаптивной версткой.

## Особенности

- Дуговое расположение карточек с настраиваемыми параметрами
- Плавные hover-эффекты и анимации
- Полностью адаптивный дизайн (мобильные, планшеты, десктоп)
- Темная цветовая схема
- Оптимизированная производительность

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- Vite
- shadcn/ui

## Установка

```bash
npm install
npm run dev
```
